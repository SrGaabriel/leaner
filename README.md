# 🪶 leaner

A linter, formatter and whole-program dead code eliminator for Lean 4.

# Installation

Freshly built versions of leaner are available on GitHub releases. You can download the latest release from [here](https://github.com/SrGaabriel/leaner/releases).

Alternatively, you can build and install leaner from source using the following commands:

```bash
git clone https://github.com/SrGaabriel/leaner.git
cd leaner
lake build
lake run install
```
